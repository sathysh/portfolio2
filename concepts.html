Concepts Learned

Dynamic Memory Allocation

Dynamic memory allocation allows a program to allocate memory at runtime. This is essential in scenarios where the size of data structures (like arrays or linked lists) is unknown at compile time. Common functions for dynamic memory allocation in C/C++ include malloc(), calloc(), realloc(), and free().

Stacks & Queues

Stacks and queues are linear data structures used to manage collections of elements. A stack follows a Last-In-First-Out (LIFO) principle, while a queue follows a First-In-First-Out (FIFO) principle. These structures are widely used in algorithms and process management.

Linked Lists

A linked list is a linear data structure where elements are not stored at contiguous memory locations. Each element, or node, points to the next one, allowing for dynamic insertion and deletion. Linked lists come in various forms like singly linked, doubly linked, and circular linked lists.

Trees

Trees are hierarchical data structures consisting of nodes, with a single root node and subnodes that form a parent-child relationship. Common tree types include binary trees, binary search trees, and AVL trees. Trees are used in applications like database indexing and file system hierarchies.
